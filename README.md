🔄 Inversor de Nomes

Este projeto consiste em um código simples que recebe uma sequência de nomes e retorna essa sequência invertida.

🚀 Funcionalidade
Entrada: lista de nomes
Saída: lista com a ordem invertida
💻 Exemplo

Entrada:

Ana, João, Carlos

Saída:

Carlos, João, Ana
🛠️ Tecnologias
Linguagem utilizada no código
